### Langkah-langkah:

1. **Buka file README.md** di repository GitHub
2. **Klik tombol "Raw"** (pojok kanan atas)
3. **Select All** (Ctrl+A / Cmd+A)
4.  **Copy** (Ctrl+C / Cmd+C)
5. **Buka Microsoft Word** (blank document)
6. **Paste** (Ctrl+V / Cmd+V)
7.  **Format dokumen:**
   - Pilih semua teks (Ctrl+A)
   - Ubah font menjadi: **Times New Roman, 12pt**
   - Atur line spacing: **1.5**
